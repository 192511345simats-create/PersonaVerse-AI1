# PersonaVerse AI

## Beyond resumes. An AI that understands your journey.

### Problem Statement
Students' certificates, resumes, GitHub repositories, hackathons, internships, and achievements are scattered across different platforms. Existing storage systems save files but don't understand a person's journey.

### Solution
PersonaVerse AI is an AI-powered Digital Identity Platform that organizes scattered records into a structured, searchable, and intelligent career knowledge repository.

## Features
- AI Document Upload
- OCR Text Extraction
- Smart Career Timeline
- Career DNA Analysis
- Digital Reputation Score
- AI Career Prediction
- AI Chat Assistant

## Tech Stack
- React
- FastAPI
- MongoDB
- Firebase Authentication
- Gemini API
- OCR
